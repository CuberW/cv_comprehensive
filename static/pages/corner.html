Harris Corner Detection

Harris 角点检测

Algorithm Ready

Corners are pixels where intensity changes significantly in all directions. Harris constructs a "structure tensor" and uses its eigenvalues to classify each pixel as corner, edge, or flat region.

Original -> Grayscale → Ix & Iy → Ixx,Iyy,Ixy → Gaussian Smooth → Response R → NMS -> Corners
