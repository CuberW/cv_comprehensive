实验口径

OpenCV 是 C++ / SIMD / 多线程基线，不和手写 NumPy 放在同一个工程复杂度里比。

`optimized v1` 主要吃可分离卷积收益，`optimized v2` 继续把 Sobel 并行化，并把 NMS / hysteresis 放进 Numba。

朴素 Python 的四层 for 不在大图硬跑到底，超过安全尺寸就用 256px 实测值按像素数外推，避免页面数据生成卡死。
