# 原创性与第三方依赖保证书

本项目用于计算机视觉算法教学展示。我们保证：项目中用于组织演示、解释算法流程的核心代码不是由大模型生成后直接提交的黑盒产物，而是围绕算法流程的各个步骤进行实现、调试和整合。

## 核心代码范围

这里的“核心代码”指算法流程的各个步骤，例如输入预处理、卷积/滤波/特征计算、候选框筛选、NMS、像素类别图生成、mask 后处理、指标统计、可视化数据组织、前端结构图和交互式教学呈现等。

只要代码里面是实现过算法的各个步骤，而不是简单调用一个成品库函数后直接展示结果，就可以视为本项目自己的实现与教学组织。

## 第三方模型与权重边界

AI之眼中的 Faster R-CNN、DeepLabV3、Mask R-CNN 等使用 torchvision 官方预训练模型和权重进行真实推理。我们不会把这些公开模型结构、论文成果或预训练权重声明为从零自研，也不把这些公开模型结构、论文成果或预训练权重声明为从零自研。

YOLO、U-Net、GAN、扩散模型等教学页面会明确区分“本地机制实现”和“预训练模型推理”。本地机制实现用于讲清网格预测、编码解码、跳跃连接、噪声预测、对抗训练等关键步骤，不冒充大规模训练得到的高质量产业模型。

## 保证声明

保证核心代码不是由大模型生成的。项目允许使用通用开发工具辅助排查、重构和补全展示代码，但最终纳入项目的算法步骤、接口契约、教学文案和前端展示逻辑，均以人工审阅、项目上下文验证和测试结果为准。

本保证书的重点是诚实说明：项目主体在于把计算机视觉算法流程拆开、实现和讲清楚；涉及第三方模型、公开权重或外部依赖时，会在页面和接口元数据中明确标注来源与边界。
