Object Detection

Algorithm Ready

Object detection answers two questions simultaneously: "what is it?" and "where is it?". YOLO divides the image into a grid, with each cell predicting bounding boxes and class probabilities in a single forward pass.

Input -> Grid Division → Feature Extract → BBox Regression → NMS -> Detection Result
