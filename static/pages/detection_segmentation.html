为什么这些网络能做检测和分割

R-CNN 与 U-Net：一个先框出物体，一个逐像素还原区域

打开完整谱系
R-CNN / Faster R-CNN 系列 用于目标检测，也能作为 Mask R-CNN 的实例分割基础
输入图像 一张普通图片，里面可能有很多物体
候选区域 先猜“哪里可能有物体”，产生很多 region proposals
CNN 特征 把每个候选区域变成可分类的特征向量
分类 + 框回归 判断是什么物体，并把框的位置修准
检测结果 类别、置信度、边界框

R-CNN 的核心功能不是“直接分割图片”，而是把整张图拆成一批可能含有物体的区域，再对每个区域做识别和定位。Faster R-CNN 把“候选区域”这一步也做进网络里；Mask R-CNN 在这个基础上再加 mask 分支，所以它能进一步做实例分割。

U-Net / FCN / DeepLab 系列 用于语义分割，目标是输出像素级类别图

U-Net 的核心功能是像素级分类。左边编码器逐步压缩图像，获得“这是什么区域”的语义理解；右边解码器把分辨率恢复回来；跳跃连接把早期的边缘细节送回后面，所以输出能更贴近物体轮廓。

任务差异

任务输出形式主流路线适用场景局限
目标检测类别 + 置信度 + 边界框Faster R-CNN、RetinaNet、FCOS、YOLO定位、计数、跟踪入口、粗粒度场景理解只能给矩形框，不能表达精细边界
