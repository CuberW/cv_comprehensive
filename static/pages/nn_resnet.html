ResNet-50 · 残差网络

ImageNet 冠军 152层可训练 Grad-CAM

网络结构

残差连接 F(x)+x 让梯度直通浅层 瓶颈块 1×1降维→3×3→1×1升维 BatchNorm 稳定训练，允许更大学习率 Grad-CAM 梯度加权激活图定位决策
Input224²×3 → Conv 7×764, /2 → MaxPool/2 → Bottleneck×3256, /4 → Bottleneck×4512, /8 → Bottleneck×61024, /16 → Bottleneck×32048, /32 → AvgPool → FC1000
⤴ 每个Bottleneck含跳跃连接：输出 = F(x) + x
为什么残差连接是革命性的？
在ResNet之前，20层以上的网络训练会梯度消失。ResNet的解法是让网络学习"残差"：不做H(x)，做F(x)=H(x)-x，通过跳跃连接把输入x直接加到输出上。梯度可以通过加法门直接流到浅层，不再被乘法逐层衰减。

瓶颈块结构：1×1卷积先把256维降到64维→3×3卷积在低维空间做特征提取→1×1卷积再升回256维。这个"窄-宽-窄"设计比两个3×3卷积参数少17倍。
