Torchvision Pretrained Studio

AI之眼：同一张图的三种视角

目标检测回答“是什么、在哪里”，语义分割回答“每个像素是什么类别”，实例分割继续回答“这是同类里的哪一个个体”。Faster R-CNN、FCN/DeepLab、Mask R-CNN 来自真实 torchvision 预训练模型；YOLO 和 U-Net 以本地机制实现展示关键数据流，结果同样由 Python 后端计算。

目标检测Faster R-CNN、RetinaNet、FCOS 等模型输出类别、置信度和边界框。
语义分割FCN、DeepLabV3、LR-ASPP 输出每个像素的类别标签和置信度。
实例分割Mask R-CNN 在检测框基础上为每个目标生成独立 mask。

运行设置

选择任务后可切换该任务的不同 torchvision 官方预训练模型。

未选择图片，将使用内置示例图。

首次运行未缓存模型时，torchvision 会下载官方权重。权重不放入仓库，可运行 python prepare_ai_eye_assets.py --all 预先准备。

后端真实模型推理中，首次加载权重可能需要一些时间。
上传图片或直接使用示例图，然后运行模型。结果会按“三视角总览 + 每一步中间结果”展开。

三种任务怎么比较

任务输出主流模型适用场景局限
目标检测类别 + 置信度 + 边界框Faster R-CNN / RetinaNet / FCOS / YOLO计数、定位、跟踪入口、粗粒度场景理解边界是矩形，不能表达精细轮廓
语义分割每个像素一个类别FCN / DeepLabV3 / SegFormer / Mask2Former道路、天空、人体、医学或遥感区域理解同类目标会合并，不能区分第几个个体
实例分割类别 + 框 + 每个实例 maskMask R-CNN / YOLO-seg / YOLACT / Mask2Former需要独立物体轮廓、遮挡分析、精细抠图计算更重，依赖训练类别和 mask 质量
